关于系统

地质公园疑似违法图斑管理系统

版本 1.0.0

系统简介

本系统是用于管理地质公园内疑似违法图斑的Web应用，提供图斑录入、整改跟踪、统计分析和事件管理等功能。

技术栈
  • 后端：Python + Flask
  • 数据库：SQLite + SQLAlchemy
  • 前端：Bootstrap 5 + jQuery
  • 图表：Chart.js
主要功能
  • 图斑信息管理（新增、编辑、删除、导入导出）
  • 整改跟踪记录
  • 事件关联管理（卫片下发、专项督查等）
  • 统计分析与可视化
  • 地图位置展示
系统信息
开发环境 Python 3.8+ / Flask 2.3.0
数据库 SQLite
前端框架 Bootstrap 5.3
